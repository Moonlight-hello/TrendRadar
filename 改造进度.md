- ✅ `data_fetch_log` 表（采集日志）
- ✅ 数据去重（按source_id + date + product）
- ✅ 查询接口（按产品/类别/日期范围）
- ✅ 统计信息接口

**数据库结构**：
```sql
CREATE TABLE commodity_data (
    id INTEGER PRIMARY KEY,
    source_id TEXT NOT NULL,
    date DATE NOT NULL,
    value REAL NOT NULL,
    category TEXT NOT NULL,     -- price/inventory/production/macro
    data_type TEXT,              -- spot/future/exchange
    product TEXT NOT NULL,       -- steel/copper/aluminum
    unit TEXT,
    region TEXT DEFAULT 'China',
    metadata TEXT,               -- JSON
    created_at TIMESTAMP
);
```

## 🔧 当前可用功能

### 1. 数据获取

```python
from trendradar.data import DataSourceManager

# 创建管理器
manager = DataSourceManager("config/data_sources.yaml")

# 获取单个数据源
data = manager.fetch_one("rebar_future_price")

# 批量获取
all_data = manager.fetch_all()
```

### 2. 数据存储

```python
from trendradar.storage.data_storage import DataStorage

# 创建存储
storage = DataStorage("output/commodity_data.db")

# 保存数据
storage.save(data_points)

# 查询数据
results = storage.query(
    product="steel",
    category="price",
    start_date=start,
    end_date=end
)

# 获取最新数据
latest = storage.get_latest(product="steel", category="price")

# 统计信息
stats = storage.get_statistics()
```

   ```python
   # trendradar/jobs/data_collector.py
   class DataCollectorJob:

           # 获取所有数据源
           # 保存到数据库
           # 记录日志
   ```

2. **集成到scheduler**
   ```python
   # 在 trendradar/__main__.py 中
   if DATA_COLLECTION_ENABLED:
       scheduler.add_job(
           data_collector.run,
           'interval',
           hours=1
       )
   ```

3. **配置更新频率**
   ```yaml
   # config/data_sources.yaml
   rebar_future_price:
       update:
           frequency: "hourly"
           schedule: "0 * * * *"
   ```
