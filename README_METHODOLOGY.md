## 🎯 核心创新

### 从"新闻监控"到"深度分析"

**TrendRadar v1.0**（当前）：
```
新闻监控 → 简单AI摘要
```

**TrendRadar v2.0**（方法论驱动）：
```
新闻 + 数据 → 结构化Prompt → AI深度分析 → 系统性认知
              ↑
         知识沉淀方法论
```

### 关键差异

| 维度 | v1.0 | v2.0（方法论驱动） |
|------|------|-------------------|
| 分析结构 | 随机、零散 | 系统化（6大章节） |
| 数据使用 | 可选 | **强制要求** |
| 认知深度 | 表面 | 深入本质（成本、产业链） |
| 可复用性 | 低 | 高（模板化） |
| 回答问题 | "铝价涨了" | "多少电？多少成本？为什么涨？" |

### 生成的Prompt（片段）
```markdown
你是一位资深的aluminum行业分析师。请基于以下数据和新闻，
完成行业宏观认知分析。

# 📊 数据输入
## 最近12个月数据

### 价格数据
- 沪铝期货均价：15,200元/吨
- 氧化铝现货价：2,400元/吨
...

### 产量数据
- 电解铝月产量：340万吨/月
- 同比增长：+3.2%
...

# 🎯 分析要求
请按照以下框架进行分析（要求：每个结论必须引用具体数据支撑）：

## 1. 行业盈利模式（10%）
- 这个行业靠什么赚钱？
- **数据要求**：必须包含行业平均利润率、龙头企业利润率对比

## 2. 产业链结构（20%）
- 上游：原料是什么？来自哪里？占成本多少？
- **数据要求**：上游原料成本占比（电力XX%，氧化铝XX%）

...（6大章节完整要求）

# 📝 输出格式
- 总字数：1500-2000字
- 必须包含数据速查表
- 每个结论后标注数据来源
- 最后总结"3个核心认知"
```

```markdown

电解铝靠"加工费"赚钱：铝价 - (氧化铝+电力+其他)
- 行业平均毛利率：12-15%（Wind，2024年）
- 龙头企业（中国宏桥）：15.2%（2024年报）

## 2. 产业链结构
铝土矿（几内亚45%）→ 氧化铝 → 电解铝 → 铝材加工
成本结构：
- 电力：13,500度/吨（37.4%）
- 氧化铝：1.93吨/吨（36.6%）
...
```

## 💡 关键价值

### 1. 回答细节问题 ✨
**传统方式**：
```
Q: "电解铝需要多少电？"
A: "很多"
```

**方法论驱动**：
```
Q: "电解铝需要多少电？"
A: "13,500度/吨，占成本37.4%，
    新疆电价0.28元/度，成本3,780元，
    山东电价0.45元/度，成本6,075元，
    差价2,295元/吨（20%）"
```

### 2. 强制数据支撑 🎯
每个结论后必须标注：
- 数据来源（国家统计局、SMM、公司年报）
- 具体数值（不能用"较高"、"很多"等模糊词）
- 如数据缺失，明确标注"数据待补充"

### 3. 系统性认知 📚
不是孤立分析，而是：
- 纵向：产业链（上游→中游→下游）
- 横向：区域对比（新疆vs山东）
- 时间：历史→现在→未来
- 空间：宏观→中观→微观
