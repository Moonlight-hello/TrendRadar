**Demo回答**：
```markdown
# 成本结构深度拆解

生产1吨电解铝需要：
- 电力：13,500度（成本4,725元，占37.4%）
- 氧化铝：1.93吨（成本4,632元，占36.6%）
- 预焙阳极：0.55吨（成本1,045元，占8.3%）
- 总成本：12,646元/吨（山东），11,701元/吨（新疆）

区域差异：
- 新疆电价0.28元/度 → 总成本11,701元
- 山东电价0.45元/度 → 总成本13,996元
- 成本差：2,295元/吨（20%）

关键发现：
1. 电力是王道（占成本37%）
2. 新疆成本优势巨大（低20%）
3. 90%新增产能都在西北
```

## 💡 关键洞察

### 洞察1: Prompt设计 = 认知框架设计

好的Prompt不是简单的问题，而是：
- 结构化的分析框架（6大章节）
- 强制性的数据要求（必须标注来源）
- 明确的输出格式（1500字 + 速查表）
- 可复用的模板（跨行业）

### 洞察2: 数据 + Prompt = 系统性认知

不是孤立的数据，而是：
- 纵向：产业链（上游→中游→下游）
- 横向：区域对比（新疆vs山东）
- 时间：历史→现在→未来
- 空间：宏观→中观→微观

### 洞察3: 方法论可工程化

知识沉淀方法论（定性）→ YAML配置（工程化）→ 自动化执行
