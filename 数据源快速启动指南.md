## 📋 第三步：配置数据源

```bash
# 1. 复制示例配置
cp config/data_sources_example.yaml config/data_sources.yaml

# 2. 编辑配置文件（可选）
# 启用或禁用特定数据源
vi config/data_sources.yaml
```

**配置示例**：

```yaml
data_sources:
  # 螺纹钢期货 - 启用
  rebar_future_price:
    name: "螺纹钢期货价格"
    type: "akshare"
    enabled: true  # ← 改为true启用
    function: "futures_main_sina"
    params:
      symbol: "RB0"

  # 锌期货 - 禁用
  zinc_future:
    name: "锌期货价格"
    type: "akshare"
    enabled: false  # ← 改为false禁用
```

## 🚀 第四步：使用数据源管理器

创建一个简单的测试脚本：

```python
# test_data_sources.py
import sys
sys.path.insert(0, '.')

from trendradar.data.source_manager import DataSourceManager

# 1. 创建数据源管理器
manager = DataSourceManager("config/data_sources.yaml")

# 2. 列出所有数据源
print("可用数据源:")
for source in manager.list_sources():
    status = "✓" if source['enabled'] else "✗"
    print(f"  {status} {source['id']}: {source['name']}")

# 3. 获取螺纹钢期货数据
print("\n获取螺纹钢期货价格:")
data = manager.fetch_one("rebar_future_price")

for dp in data[-5:]:  # 显示最近5条
    print(f"  {dp.date.strftime('%Y-%m-%d')}: {dp.value:.2f} 元/吨")

# 4. 获取所有启用的数据源
print("\n批量获取所有数据:")
all_data = manager.fetch_all()

for source_id, data_points in all_data.items():
    if data_points:
        latest = data_points[-1]
        print(f"  {source_id}: {len(data_points)}条数据, 最新={latest.value:.2f}")
```

运行：
```bash
python test_data_sources.py
```

## 🔧 常见问题

### Q1: AKShare安装失败

```bash
# 方法1：使用国内镜像
pip install akshare -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q2: 获取数据失败

**原因1：网络问题**
- 检查网络连接
- 尝试使用代理

**原因2：接口更新**
- AKShare接口可能更新
- 访问文档查看最新接口：https://akshare.akfamily.xyz/

**原因3：数据源暂时不可用**
- 某些数据源可能维护中
- 稍后重试

### Q3: 如何添加新的数据源？

**步骤1：在配置文件中添加**

```yaml
# config/data_sources.yaml
data_sources:
  # 新增数据源
  new_data_source:
    name: "新数据源"
    type: "akshare"
    enabled: true
    function: "akshare_function_name"  # AKShare函数名
    params:
      param1: "value1"
    data_category: "price"
    product: "steel"
```

**步骤2：测试**

```python
manager = DataSourceManager("config/data_sources.yaml")
data = manager.fetch_one("new_data_source")
print(data)
```

## 📈 集成到TrendRadar主流程

### 方案1：定时采集数据

```python
# trendradar/jobs/data_collector.py
from trendradar.data.source_manager import DataSourceManager
from trendradar.storage.data_storage import DataStorage

def collect_data_job():
    """定时数据采集任务"""
    manager = DataSourceManager("config/data_sources.yaml")
    storage = DataStorage("output/data.db")

    # 获取所有数据
    all_data = manager.fetch_all()

    # 保存到数据库
    for source_id, data_points in all_data.items():
        storage.save(data_points)

    print(f"✓ 采集完成，共{sum(len(d) for d in all_data.values())}条数据")

# 添加到调度器
```
