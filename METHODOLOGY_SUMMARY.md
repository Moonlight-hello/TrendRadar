## 🎯 关键创新：方法论驱动的Prompt体系

### 传统方式 vs. 方法论驱动

| 维度 | 传统方式 | 方法论驱动（本方案） |
|------|---------|---------------------|
| **分析思路** | 随机、零散 | 阶段性、递进式 |
| **数据使用** | 简单罗列 | 结构化关联（上下游、成本拆解） |
| **结论依据** | 可能缺乏数据支撑 | **强制要求数据支撑** |
| **可复用性** | 每次重新思考 | 模板化、可复制 |
| **深度** | 表面现象 | 深入本质（成本、产业链） |

### 核心优势

1. **结构化Prompt模板**
   - 每个阶段有明确的分析框架
   - 强制要求数据完整性
   - 可跨行业复用

2. **数据驱动**
   - 从新闻监控 → 新闻+数据综合分析
   - 自动关联上下游数据
   - 自动计算成本结构

3. **符合认知规律**
   - 宏观 → 中观 → 微观
   - 由浅入深
   - 持续跟踪

## 💡 关键洞察

### 1. 方法论驱动 ≠ 简单的数据堆砌

**错误做法**：
```
产量：340万吨
价格：15,200元
库存：85万吨
```

**正确做法**（方法论驱动）：
```
产量340万吨（+3.2%），但需求350万吨（+5.1%），
供需缺口10万吨/月导致库存下降至85万吨（历史35分位），
低库存支撑价格上涨至15,200元（+6.8%）。

结论：供需偏紧是价格上涨的根本原因，而非投机炒作。
```

### 2. 上下游关联 = 产业链思维

不是孤立看电解铝，而是：
- 向上追溯：铝土矿 → 氧化铝 → 电力成本
- 向下追踪：汽车、家电、建筑需求变化
- 横向对比：不同地区成本差异

### 3. 阶段性认知 = 符合学习规律

不是一次性灌输所有信息，而是：
- Week 1: 宏观认知（能聊10分钟）
- Week 2-3: 公司研究（形成判断）
- 持续: 微观跟踪（捕捉变化）
